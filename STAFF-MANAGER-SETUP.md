# Staff Manager Portal Setup

1. Sign in to `frontend/admin/index.html` with the main admin account.
2. Open **Staff**.
3. In **Staff manager access**, create a separate login for the person who should control the full Staff section.
4. Give that person the direct page URL:
   - `frontend/staff-management.html`
5. That user can sign in there and manage:
   - Staff profiles
   - Service assignment per staff member
   - Attendance
   - Work logs
   - Payroll / adjustments

Notes:
- The main admin account can still use the same standalone staff portal too.
- If you deactivate a staff manager account, the login stops working.
